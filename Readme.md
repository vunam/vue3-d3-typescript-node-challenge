Requirements 
Set variable `NEO4J_AUTH` in the `.env` to `neo4j/any-password-here`. Install neo4j or run the following docker command:

```
  docker run --name neo4j -p7474:7474 -p7687:7687 -d --env-file .env neo4j:latest
```
